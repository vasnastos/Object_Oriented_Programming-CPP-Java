ΟΔΗΓΙΕΣ ΕΓΚΑΤΑΣΤΑΣΗΣ JDK ΚΑΙ ECLIPSE IDE



Εγκατάσταση jdk

1.Λήψη του development kit από τον ακόλουθο σύνδεσμο:JDΚ

2.Τρέχουμε το .msi αρχείο.

3.Διαλέγω από την επιλογή set JAVA_HOME variable την πρώτη λειτουργικότητα(will be installed on local hard drive).


4.Πατάω το κουμπί next.


5.Τσεκάρουμε την επιλογή I accept the terms in the license Aggriment και πατάμε next.


5.Πατάμε το κουμπί next.


6.Πατάμε το κουμπί Install


  • Ελέγχουμε αν το development kit έχει εγκατασταθεί στον υπολογιστή μας,εκτελώντας στην γραμμή εντολών την εντολή java -version
  • Ελέγχουμε αν ο compliler για την γλώσσα java έχει εγκατασταθεί σωστά,εκτελώντας στην γραμμή εντολών την εντολή javac -version



    • Αν εμφανίζονται τα παρακάτω μυνήματα έχετε ολοκληρώσει την Εγκατάσταση του development kit και του compiler της java,και μπορείτε να γράψεται και να μεταγλωττίσετε ένα πρόγραμμα σε java.

    Εγκατάσταση του Eclipse IDE

    1.Λήψη του Eclipse Installer από τον ακόλουθο σύνδεσμο:Eclipse

    2.Πατάω το κουμπί run για εκτέλεση του .exe αρχείου


    3.Επιλέγω το πακέτο για ανάπτυξη λογισμικού σε java(Eclipse IDE for java developers).


    4.Επιλέγω φάκελο εγκατάστασης για το eclipse(Συνίστανται ο φάκελος program files(Αρχεία εφαρμογών)).Επίσης τσεκάρω τις επιλογές create start menu entry και create desktop shortcut.


    5.Εκκινώ to eclipse επιλέγωντας τον φάκελο που θα αποθηκευτεί ο αντίστοιχος source folder του project που θα δημιουργήσω.
