Οδηγίες εκτέλεσης του προγράμματος

1.Στο φάκελο που είναι αποθηκευμένα τα αρχεία του πηγαίου κώδικα αποθηκεύστε τα ακόλουθα dll
1.* wxbase314u_gcc_custom.dll
2.* wxmsw314u_core_gcc_custom.dll
2.Μεταγλώττιση κώδικα:
:: g++ Surfer.cpp Matrix.cpp gui.cpp -IC:\\wxWidgets-3.1.4\\include -IC:\\wxWidgets-3.1.4\\lib\\gcc_dll\\mswu -LC:\\wxWidgets-3.1.4\\lib\\gcc_dll -lwxmsw31u_core -lwxbase31u -o runnable
:: g++ Surfer.cpp Matrix.cpp gui.cpp `wx-config --cxxflags --libs` -o runnable

3.Εκτέλεση:
:: runnable
:: ./runnable
